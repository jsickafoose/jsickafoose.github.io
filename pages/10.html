Fundamentals of Robot Kinematics and Dynamics - ECE 10

In my Robot Kinematics and Dynamics course, I was required to implement several controller programs using Python inside of a simulated environment called "WeBots". In this course, the robots and most of the environments were provided and I was expected purely to write code to control the robots.

Unfortunately, the actual PDF write-ups for this course were required to follow a strict format which resulted in most of them not being self-documenting. I have chosen to include the report for the final project in the class because it is the most comprehensive and I worked on it as part of a small team. The rest of the code as well as the Markdown README's can still be found here .
