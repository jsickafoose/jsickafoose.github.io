Analog Electronics - ECE 171

In my Analog Electronics course, I was constantly building and analyzing circuits then charting the results. Being able to build, make changes to a circuit and then measure the results was an invaluable skill for working on my Senior Design Project. A selection of the my lab write-ups can be found below. In the first one I was tasked with calculating different values across a BJT circuit and verifying the results experimentally. This required entering several different values in a spreadsheet and plotting the results using MATLAB.

In the following lab I was also tasked with calculating several valeus and experimentally verifying the results. In this lab, I was expected to find these values across various MOSFET circuits.
