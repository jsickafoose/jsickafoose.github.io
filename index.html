Senior Design Project: Tactile Stimulation

Simulated virtual reality textures using transcutaneous electrical nerve stimulation. Created using circuit design and modeling, iterative prototyping, additive manufacturing, and microcontroller programming for precise waveforms implementation.

Sensing and Sensor Technologies

Integrated sensors into a digital system using the C language running on a microcontroller. Sensor signal conditioning and processing utilized filters, amplification, and/or analog-to-digital conversion.

Analog Electronics

Analysis and synthesis of linear and nonlinear electronic circuits containing diodes and transistors. Biasing, small signal models, frequency response, and feedback. Operational amplifiers and integrated circuits.

Fundamentals of Robot Kinematics and Dynamics

Utilized the theory and application of mathematical models to analyze the kinematics and dynamics of robot mechanisms or their components using vector algebra, differential equations, and computer simulations. Also covered robot vehicle kinematics, robot arm kinematics, and robot dynamics with computational examples and problems.

Logic Design

Boolean algebra, logic minimization, finite-state machine design, sequential circuits, common logic elements, programmable logic devices, and system level design. Also covered the electrical behavior of circuits including three state outputs, propagation delay, logic levels, and fanout.

Small-Scale UAV Theory and Practice

Studied the technologies involved with the modeling and simulation of small-scale unmanned aerial vehicles (UAVs) with an emphasis on control applications, from low-level flight stabilization to higher level path planning and vision-based control.

Microcontroller System Design

Focused on the design and use of microcontroller-based embedded systems, specifically addressing issues of low-level functionality, direct manipulation of input/output using various specialized peripheral sets, and multiple communications protocols. Fully custom APIs were written to interact with the raw hardware used to implement the programs.

Renewable Energy Sources

Introduction to energy sources and storage with a special emphasis on renewables as part of smart grids.

Personal Hobbyist Works

Collection of images from personal projects worked on outside of any organization. These projects were worked on with an emphasis on speed in order to maximize the time that was available. This resulted in a lack of complete documentation.
